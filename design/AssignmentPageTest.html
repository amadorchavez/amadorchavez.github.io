Module 1.8 Review

In this module we learned the 5 universal principles of design:

  • Contrast
  • Consistency
  • Alignment
  • Subtlety
  • Readability

We also covered several of the individual design elements that can be adjusted to achieve these principles including color, typography, size, and white space.

You should now be able to apply these principles to any design you create. In the next module, we will cover the specifics of using PowerPoint to implement these principles.
