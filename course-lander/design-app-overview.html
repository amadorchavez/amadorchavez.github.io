Design Tool Overview

The goal is to create a native app with common design tools, including:

  • Contrast Checker: This will allow the user to input two colors and will calculate whether they contrast enough based on WCAG standards.
  • Color Extractor: This will allow the user to import an imag and then extract the most common colors from it to create a color palette
  • Duotone Generator: This will allow the user to import images and apply a duotone filter. Example
  • Color Blindness Simulator: Allows users to import mockups and experience how a color-blind person would see it.
  • Style Guide Generator:
  • Font Size Proportion Generator: Generates proportion based on a consistent, proportional ratio.
